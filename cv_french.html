PROJETS
PrintWizard - Immediate tracing for smoother debugging and code exploration
Spring 2024
Thèse de Master réalisée au SYSTEMF, EPFL.

Conception de PrintWizard, un débogueur d'historique. PrintWizard enregistre l'historique complet d'un programme Java et vous permet ensuite d'explorer interactivement ce qui s'est passé. Il vous aide à déboguer les problèmes difficiles à résoudre avec un débogueur standard.

Le projet se compose de deux parties. Un backend Java, un plugin pour le compilateur Java qui modifie le programme au moment de la compilation. Ensuite, lorsqu'il est exécuté, le programme enregistre son historique complet dans un journal d'événements. Et une interface utilisateur graphique implémentée en Javascript et HTML qui permet l'exploration interactive de l'historique du programme.

Query optimization for hybrid query execution
Fall 2022
Projet de semestre réalisé au, EPFL.

Conception du support pour les plans exécutés simultanément sur le GPU et le CPU dans le planificateur de requêtes de Proteus, un moteur de base de données développé à l'EPFL. Le résultat a été une accélération moyenne de 10% de l'exécution des requêtes. Le travail a consisté à concevoir un nouvel algorithme efficace pour paralléliser les plans entre le CPU et à modifier le planificateur de requêtes implémenté en Java.
